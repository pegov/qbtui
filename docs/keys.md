# Keys

j Down  - Move down
k Up    - Move up
h Left  - Move left
l Right - Move right
q Esc   - Back/Quit
? F1    - Help

## Torrents

s       - resume
p       - pause
i       - info
o Enter - open file/folder
O       - open folder in the default file manager
c       - categories
x       - delete
X       - delete files
r       - reload
/       - search

## Search

Enter - apply
Esc   - discard
